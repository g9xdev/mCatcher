Almost there — install the recorder helper

Media Catcher records live video and audio to real files using a small native helper that drives ffmpeg. Install it once and you're set.

1. Download & run the installer

Download the helper installer

Windows · MediaCatcherHostSetup.exe · installs Python + ffmpeg if they're missing and registers the helper with Firefox.

2. Finish up

  1. Run the downloaded MediaCatcherHostSetup.exe and let it complete.
  2. Restart Firefox (or click Re-check below) so it picks up the helper.

Where “Save” puts recordings (native helper only). Blank = your Downloads folder.

Filters out tip sounds, avatars, and preview clips. 0 = show everything. Never affects HLS/DASH streams.

Re-encodes each finished recording to H.265 after saving, then deletes the H.264 original. Uses your GPU when available. Needs the native helper.

Updates

Drop a new media_catcher-*.zip (and optional media-catcher-host.zip) in the folder below, then check. The helper installs it and offers to restart Firefox (restoring your tabs). Requires the native helper; persistent auto-load needs Firefox Developer/Nightly with unsigned add-ons allowed.

The helper watches the folder via OS change events (no polling) and checks GitHub Releases for new versions, then offers to install + restart when a newer build lands.
